# Innovation Center POC

Frontend proof of concept for the MOMAH Innovation Center platform.

## Included

- Bilingual Arabic and English interface
- RTL/LTR switching
- Light and dark theme toggle
- Presentation-ready pages for:
  - Home
  - Challenges
  - Challenge Details
  - Submit Idea
  - Admin Dashboard
  - Matchmakers
- Mocked challenge, evaluation, and partner-matching data based on the BRD handoff

## Run locally

1. Install Node.js 18+.
2. Run `npm install`
3. Run `npm run dev`

## Notes

- This is intentionally frontend-only for POC/demo use.
- Content is structured so the mock data can later be replaced with API-driven data.
